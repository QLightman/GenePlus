Instruction

Input file format

The file which contain the data should obey the following format, 

            
 element1 element2 value 
 element1 element3 value 
 element2 element1 value 
 element3 element1 value 
 ...
You can consult demo1.txt as a example.

Usage

1.Load file via choose input file. The default cluster algorithm is k-means, you can adjust to spectrual cluster via spectrum.
2.When mouse hover one of the cluster, it will display the elements inside the cluster in the left viewport, by choosing one of the element, it will display its most related elements and interactions, you can adjust the nodes number and minimum interaction value via the top-left input window, you can always return to the main view by clicking return.
3. By double click one the cluster(zoom-in), it will reveal the hierarchy structure of this cluster, by click back(zoom-out) will return to the previous hierarchy. Together, it will help you navigate through different graph hierarchies. 4. By clicking save.file you can get a file with all the elements listed in your chosen cluster. So you can do further analysis such as gene ontology analysis. Also, by clicking save.svg you can get a svg file of your current view.
